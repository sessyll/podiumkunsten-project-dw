# Data Engineering Project: Automated Visualization with Datawrapper

This project demonstrates data manipulation and visualization capabilities developed during the 2022 Bit Academy final assignment. 

Originally created using Seaborn for static plotting, the pipeline has been upgraded to automate the visualization process using the **Datawrapper API**.

## Key Features
- Automated data processing and cleaning.
- Seamless integration with Datawrapper for interactive visualizations.
- [View the Datawrapper Visualization - https://datawrapper.dwcdn.net/zszNI/1/

## Tech Stack
- Python (Data Manipulation & API Integration)
- Datawrapper API
